변수에 기본값 할당 Nullish coalescing operator -ES11

변수에 기본값을 설정하고 싶을 때, 논리연산자 OR(||)을 사용하는 것이 일반적이었습니다.

이러한 사용은 자바스크립트의 false값 (0, ‘’, NaN, null, undefined) 모두를 Guest로 할당하기 때문에 잠재적 버그 가능성이 높았습니다.

ES11에 추가된 Nullish coalescing operator(??)를 사용하면 좀 더 명확하게 사용할 수 있습니다.

|| 연산자는 (0, ‘’, NaN, null, undefined)를 falsy값으로 갖는 반면,

?? 연산자는 null 또는 undefined 일 때만 두 번째 피연산자를 반환함으로
0, ‘’, NaN을 유효한 값으로 사용할 수 있다.

Optional chaining과 같이 활용

Optional chaining을 사용한 후 아무 값도 찾을 수 없을 때
기본값을 지정해주기 위해 사용 가능.

중요포인트

  • OR(||)과 AND(&&) 같은 논리 연산자들과 마찬가지로, 만약 왼쪽이 null 또는 undefined가 아님이 판명되면 오른쪽 표현식은 평가되지 않는다.

  • AND (&&) 와 OR 연산자 (||)를 ??와 직접 결합하여 사용하는 것은 불가능하다. 이 경우 SyntaxError가 발생된다.

    이런 경우 우선순위를 나타내기 위한 괄호()를 작성해주면 사용가능 하다.
