return 문,재귀 함수 호출 -JavaScript

return 문 이란 함수에서 결과값을 되돌려 줄 때 사용합니다.
함수에서 return 문이 실행되면
반복문에 break문과 비슷하게 실행문이 강제 종료됩니다.

기본형
function 함수명(){
실행문;
return 데이터(값);
}

재귀 함수 호출

함수 정의문 내에서 실행문으로 함수를 다시 호출하는 것을
재귀 함수 호출 이라고 합니다.
함수를 반복문 처럼 여러번 호출하기 위해 사용합니다.

기본형
function myFnc(){
실행문;
myFnc();
}
myFnc();
